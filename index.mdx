<div className="mt-16 mb-16 max-w-5xl mx-auto px-6">
  <h2 className="text-3xl font-semibold text-gray-900 dark:text-white mb-3">Quick Start</h2>
  <p className="text-base text-gray-600 dark:text-gray-400 mb-8">Get up and running with Maxw AI in minutes</p>

  <Steps>
    <Step title="Set up your environment">
      Clone the repository and install dependencies using Bun:

      ```bash
      git clone https://github.com/maxwiseman/maxw-ai-v3.git
      cd maxw-ai-v3
      bun install
      ```
    </Step>

    <Step title="Configure environment variables">
      Copy the environment template and add your credentials:

      ```bash
      cp .env.example .env
      ```

      Required variables include database URL, authentication secrets, and AI API keys (OpenAI, Anthropic, Google).
    </Step>

    <Step title="Initialize the database">
      Push the database schema to your PostgreSQL instance:

      ```bash
      bun db:push
      ```
    </Step>

    <Step title="Start the development server">
      Launch the web and mobile applications:

      <CodeGroup>
        ```bash All Apps
        bun dev
        ```

        ```bash Web Only
        bun dev:web
        ```

        ```bash Mobile Only
        bun dev:native
        ```
      </CodeGroup>

      The web app will be available at `http://localhost:3000`.
    </Step>
  </Steps>
</div>
